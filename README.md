# AI-Agent

Чистый репозиторий для разработки ИИ-ассистента:
- Обработка данных
- Интеграция с n8n
- Генерация контента
- Работа с PostgreSQL
- Использование OpenRouter

Структура репозитория подготовлена для удобного расширения.


Ai-Agent/
│   README.md
│   .gitignore
│
├── src/        — скрипты, исходный код
├── backend/    — серверная логика, API
├── prompts/    — промты для моделей
├── docs/       — инструкции, описания
├── media/      — изображения, логотипы
├── data/       — Excel, CSV, таблицы
├── drawings/   — чертежи, схемы
├── n8n/        — JSON-экспорты и схемы n8n
└── configs/    — env, настройки, токены (НЕ пушить)
